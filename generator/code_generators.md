# Zadanie Generator Kodu

## Cel:
Celem zadania jest zapozananie się z metodami generowania kodu bez użycia AI.

## Instrukcja

- utworzyć repozytorium w github
- pobrać i skonfigurować lokalnie gita

- Napisać generator serializacji i deserializacji danych:
    - definicja danych ma być zapisana w formacie json (interface.json)
    - generator ma czytać plik json, oraz wygenerować metody serializacji oraz deserializacji danych, do formatu binarnego.
    - go generacji kodu, należy użyć metody szablonów jinja2.

- Napisć 2 programy które będą komunikować się po TCP/IP z wykorzystaniem wygenerowanej implementacji.
    - temat programu testującego jest dowolny.

- repozytorium należy przesłać do github
- spakowany główny branch należy przesłać do moodle
